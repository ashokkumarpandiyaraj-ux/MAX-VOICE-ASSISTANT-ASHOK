# 🤖 MAX Voice Assistant (Python)

MAX is a personal voice assistant built using Python.
It can listen to your voice, understand commands, perform tasks, and even respond using AI.

* 🎤 Voice Recognition (Speech-to-Text)
* 🔊 Text-to-Speech responses
* 🌐 Open websites (YouTube, Google)
* 📖 Wikipedia search
* 🎵 Play songs on YouTube
* 🧮 Perform calculations
* ⏰ Tell time
* 🌦️ Get weather updates
* 🤖 AI-powered conversation (ChatGPT integration)
* 📱 Simple GUI using Kivy
* 
## 🛠️ Technologies Used

* Python
* SpeechRecognition
* pyttsx3
* Wikipedia API
* OpenAI API
* Kivy (GUI)
* Requests

## 📦 Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/max-voice-assistant.git
```

2. Navigate to the folder:

```bash
cd max-voice-assistant
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

---



```bash
python main.py
```

---

How It Works
* Say **"Hey Max"** to activate the assistant
* Speak your command
* MAX processes the command and responds accordingly
* If command is unknown, it uses AI to reply

---


Example Commands
* "What is the time?"
* "Open YouTube"
* "Search Wikipedia for Python"
* "Play song"
* "Weather in Chennai"
* "Calculate 5 plus 10"
